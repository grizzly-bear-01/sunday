The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects, that way, programming languages can interact with the page. Web developers can manipulate the elements and content of a web page using programming languages such as JavaScript, which can dynamically modify the content and behavior of a web page in response to user interactions or other events.
It is a tree-like structure of nodes that represents the elements, attributes, and text of an HTML document. Each element on the page is represented by a node in the DOM, and nodes can have child nodes and parent nodes, making up a hierarchy of nodes.

There are 5 ways in which one can select elements in a DOM using selectors.

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()
